### 一、产品概述
**“宠物祭祀”平台的****主功能模块概览与首页布局设计****，是一款专为宠物主人提供****宠物身后关怀服务平台****。该应用以温暖、尊重生命的情感基调为核心，围绕“宠物殡葬”这一特殊需求场景，打造了一个集****上门服务、纪念馆创建、墓地选购、代扫祭奠、订单管理****于一体的综合性服务平台。**

**画面中左侧展示了12个核心功能图标，右侧手机界面呈现了APP首页的视觉结构和主要入口：包括“预约上门”、“客户自送”、“爱宠纪念馆”、“陵园墓地”等关键服务板块。整体采用暖黄色调背景搭配可爱的宠物形象（如小狗、猫咪），营造出温馨、治愈且富有情感共鸣的用户体验氛围。**

**该平台旨在帮助用户在宠物离世后，通过专业、有尊严的方式完成告别仪式，并长期保留对宠物的记忆与情感连接，满足现代人日益增长的“宠物情感陪伴”与“生命终期关怀”需求。**

### 二、核心功能
**根据图片内容，平台的核心功能模块如下：**

#### 1. **预约上门**
+ **用户可在线预约专业人员上门接取宠物遗体。**
+ **支持填写预约人姓名、电话等信息，确保服务精准对接。**
+ **提供便捷的“一键预约”流程，减少沟通成本。**

#### 2. **客户自送**
+ **允许用户自行将宠物送往指定地点进行火化或安葬。**
+ **适合希望亲自参与告别过程的家庭，增强仪式感与心理慰藉。**

#### 3. **爱宠纪念馆**
+ **用户可创建专属的线上纪念馆，用于存放宠物照片、生平故事、纪念语录等。**
+ **支持上传头像、添加备注、设置生日与祭日，形成个性化数字记忆空间。**
+ **可免费创建，体现平台的人文关怀与情感温度。**

#### 4. **陵园墓地**
+ **提供多种类型的宠物墓地选择，如“树葬”、“土葬”等。**
+ **展示真实环境图片（如森林、花海），突出自然和谐与生态安葬理念。**
+ **用户可通过“挑选墓地”按钮进入购买流程，支持在线选位与支付。**

#### 5. **我的代扫**
+ **针对无法定期前往墓地祭拜的用户，提供“代扫服务”。**
+ **用户可在平台上提交代扫请求，由工作人员代为清理、献花、焚香等。**
+ **记录每次代扫时间、操作人、联系方式，实现服务可追溯。**

#### 6. **订单管理**
+ **统一管理所有服务订单，涵盖墓地购买、升级纪念馆、礼物赠送、代扫等类别。**
+ **每条订单显示消费时间、说明、用户编号，便于查询与核对。**
+ **支持按标签分类查看（如“墓地”、“礼品”、“代扫”），提升查找效率。**

#### 7. **我创建的纪念馆**
+ **查看用户已创建的所有宠物纪念馆列表。**
+ **显示封面图、标题及简要描述，方便快速访问。**
+ **支持后续编辑或分享给亲友共同缅怀。**

#### 8. **我购买的墓地**
+ **展示用户已购得的墓地信息，包括位置、类型、状态等。**
+ **实现“虚拟+实体”的双重归属感，让用户安心寄托思念。**

#### 9. **我的预约**
+ **查看当前已预约的服务记录，包括上门时间、服务项目等。**
+ **支持修改或取消预约，灵活应对突发情况。**

#### 10. **联系我们**
+ **提供客服联系方式，支持在线咨询或电话沟通。**
+ **解决用户在使用过程中遇到的问题，提升服务响应速度。**

#### 11. **关于我们**
+ **介绍平台品牌理念、服务宗旨、团队背景等信息。**
+ **增强用户信任感，建立品牌形象。**

#### 12. **底部导航栏**
+ **包含“首页”、“订单”、“我的”三大主入口：**
    - **“首页”：浏览服务入口与推荐内容**
    - **“订单”：查看所有历史和服务记录**
    - **“我的”：管理个人信息、纪念馆、代扫等个人中心功能**
+ **设计简洁直观，满足多场景操作需求。**

### 三、产品价值
#### 1. **填补市场空白，满足情感刚需**
+ **宠物已成为许多家庭的重要成员，其离世带来的悲痛不亚于亲人离去。**
+ **传统殡葬体系缺乏针对宠物的专业服务，“宠物祭祀”平台填补了这一空白，为用户提供****有尊严、有仪式感的告别方式****。**

#### 2. **构建完整的宠物生命周期服务体系**
+ **不仅关注宠物生前的喂养与医疗，更延伸至“身后事”，实现从出生到告别的全周期关怀。**
+ **将“殡葬”转化为一种****情感延续与精神慰藉****的过程，而非单纯的处理遗体。**

#### 3. **数字化赋能，提升服务效率与透明度**
+ **所有服务均可在线预约、下单、支付，打破地域与时间限制。**
+ **订单详情清晰可见，消费记录可追溯，增强用户信任感。**
+ **代扫服务支持远程委托，解决异地用户无法亲临祭扫的痛点。**

#### 4. **创造持久的情感连接点**
+ **“爱宠纪念馆”功能允许用户永久保存宠物的记忆，成为数字时代的“电子墓碑”。**
+ **用户可以随时回访、留言、分享，形成持续的情感互动，缓解哀伤情绪。**

#### 5. **推动宠物殡葬行业规范化发展**
+ **平台通过标准化流程（如统一编号、服务说明、客服核验）提升服务质量。**
+ **引导公众正视宠物殡葬议题，促进社会对动物权利与生命尊重的认知进步。**

#### 6. **融合生态环保理念**
+ **推出“树葬”等绿色安葬方式，倡导回归自然、可持续发展的理念。**
+ **在满足情感需求的同时兼顾生态保护，体现企业社会责任。**

#### 7. **增强用户粘性与品牌忠诚度**
+ **通过“免费创建纪念馆”、“代扫服务”等高情感价值功能，建立长期用户关系。**
+ **用户一旦使用，往往会长期维护纪念馆、定期代扫，形成稳定的复购行为。**

#### 8. **拓展商业潜力与生态闭环**
+ **平台未来可扩展至宠物纪念品销售、宠物保险、宠物医院合作等周边服务，构建“宠物生命关怀生态圈”。**
+ **数据积累可用于精准营销与个性化推荐，提升商业转化能力。**

## 架构特点
### 架构特性
+ **前后端分离架构，独立开发，符合主流开发模式**
+ **前端以Vue3+Vite为主技术，AntdV为UI界面框架**
+ **后端SpringBoot3为基础，MybatisPlus为数据操作框架，Redis为缓存框架**
+ **Maven多模块管理，插件化开发，方便安装、卸载、升级，降低耦合**
+ **业务模块与API抽离，模块之间便捷引用**
+ **数据库设计精巧，字段规范、易于扩展**
+ **支持国产密码算法加解密，等保测评国产项目无压力**
+ **支持MYSQL、ORACLE、SQLSERVER、PGSQL等主流标准结构式数据库**
+ **支持达梦、人大金仓、南大通用、九有、瀚高、虚谷数据库等国产数据库**
+ **支持中创、宝蓝德、东方通等中间件**
+ **支持Windows、Linux操作系统、国产操作系统部署**

### 功能特性
+ **完善的系统基础功能，满足使用需求，避免重复造轮子**
+ **支持本地文件、阿里云文件、腾讯云文件、MINIO文件上传**
+ **支持本地邮件、阿里云邮件、腾讯云邮件发送**
+ **支持阿里云短信、腾讯云短信发送**
+ **B、C端双账号认证体系，会话治理各自独立**
+ **完善的登录日志、操作日志、异常日志等审计功能**
+ **完善的会话监控、数据源监控、系统监控等必备监控功能**
+ **支持组织机构、权限管理、定时任务、系统配置等基础功能**

### 安全特性
+ **采用SaToken轻量级 Java 权限认证框架，功能强大、学习成本低**
+ **支持登录认证、权限认证、单点登录、三方登录、OAuth2.0等认证模式**
+ **增强的RBAC权限设计，资源于接口独立授权，更加灵活**
+ **支持按钮级别细粒度独立授权，界面按钮动态展示**
+ **支持API接口注解式、路由拦截式鉴权，防止越界访问**
+ **独创的数据范围机制，每个接口都可以配置不同数据范围**
+ **支持限流防抖，防重复提交，有效阻止脏数据产生**
+ **密码、手机、身份证号等使用国密算法加密传输、加密存储，数据更安全**
+ **操作日志使用SM2进行完整性保护，满足安全审计要求**

### 界面特性
+ **Vue3 + Vite为基础，AntdV为界面UI框架，视觉风格清新简洁**
+ **精细化设计，注重界面的每一处细节，操作轻松友好**
+ **暗黑风格、经典菜单、双排菜单、多页签、目录坞、主题切换等功能应有尽有**
+ **统一的网络框架、API接口拦截框架，拿来即可上手**

+ **Node.js 14x**
+ **Java1.8**

+ **Redis 6.2.8**
+ **阿里云 OSS – 可更换其它。**
